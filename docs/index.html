BBdownEasy

基于BBDown的次世代B站下载解决方案

# 核心功能

智能下载

  • 自动检测并安装BBDown
  • 多线程高速下载
  • 智能画质选择
  • 自动更新检查

全面支持

  • 支持WEB/TV/APP/国际版API
  • 弹幕/封面/字幕下载
  • 会员专属内容下载
  • 多账号登录管理

> 使用模式

快速模式

一键下载最高画质:

  • 自动选择最佳画质
  • 默认下载弹幕和封面
  • 多线程加速

高级模式

完全自定义下载:

  • 选择API模式（Web/TV/APP/国际版）
  • 自定义画质优先级
  • 设置编码格式（HEVC/AV1等）
  • 选择是否下载弹幕、封面、字幕
  • 自定义保存路径
  • 分P选择下载

? 如何使用

1. 下载安装

从GitHub下载最新版本，解压后运行bbdownInstall.bat(或者bbdownInstall.sh)

2. 使用

运行BBdownEasy.bat(或者BBdownEasy.sh)，根据提示完成配置

  • 快速下载模式：输入视频URL即可
  • 高级下载模式：支持自定义API模式、画质优先级、编码格式等
