BBdownEasy

基于BBDown的次世代下载解决方案

# 项目简介

核心功能

  • 自动检测安装BBDown
  • 图形化交互界面
  • 多线程下载支持
  • 自动版本更新

> 功能特性

自动安装与更新

首次运行时自动检测环境

通过Chocolatey自动安装:

  • BBDown下载器
  • FFmpeg编码器

交互式下载界面

提供两种操作模式:

  • 快速下载模式
  • 高级下载模式

$ 使用说明

:: 安装步骤
bbdownInstall.bat
:: 运行方式
bdown.bat

安装步骤

  1. 双击运行 bbdownInstall.bat
  2. 系统将自动完成环境配置
